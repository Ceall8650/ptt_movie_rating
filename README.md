# PTT Movie Rating
You can get the rating of each movie that published in PTT movie group

# Environment requirement
- Node.js: v18.15.0 or later

# Tech Stack
- Next.js v14
- TypeScript 5
- Tailwind 3

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
